Tourism in India is a responsive travel booking website that allows users to explore hotels across multiple cities and estimate the total travel package cost. The website provides an integrated hotel booking system along with a transportation cost calculator that calculates travel expenses based on the number of travelers, source location, and destination city.

This project demonstrates front-end development skills using HTML, CSS, Bootstrap, and JavaScript while focusing on real-world travel booking functionality.
